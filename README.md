# 🎚️ Volume Mixer for Android

تطبيق يتيح لك التحكم في صوت كل تطبيق بشكل مستقل، مثل Windows Volume Mixer تماماً!

## ✨ المميزات

- 🎵 يعرض كل التطبيقات التي تشغل صوتاً الآن فقط
- 🪟 نافذة عائمة يمكن سحبها في أي مكان على الشاشة
- 🔊 شريط تحكم (Slider) مستقل لكل تطبيق
- 🔇 زر كتم لكل تطبيق
- ⚡ تحديث تلقائي فوري عند بدء/إيقاف أي تطبيق
- 📱 يدعم تشغيل أكثر من تطبيق في نفس الوقت

## 📥 تنزيل الـ APK

1. اذهب إلى تبويب **Actions** في هذا الـ Repository
2. اضغط على آخر Build ناجح ✅
3. نزّل ملف `volume-mixer-debug`
4. افتح الـ APK على هاتفك وثبّته

## 🔧 الإعداد

عند أول تشغيل:
1. اضغط **"تشغيل Volume Mixer"**
2. سيطلب إذن **"Display over other apps"** — وافق عليه
3. ارجع للتطبيق واضغط **"تشغيل Volume Mixer"** مجدداً
4. ستظهر النافذة العائمة 🎉

## ⚠️ ملاحظة تقنية

بسبب قيود أندرويد، التحكم في مستوى الصوت بشكل فردي لكل تطبيق يتطلب صلاحيات Root أو Shizuku.
النسخة الحالية تتيح:
- ✅ رؤية التطبيقات النشطة صوتياً
- ✅ كتم/تفعيل كل تطبيق
- ✅ واجهة Volume Mixer كاملة
- ⚠️ التحكم الدقيق في الـ % يحتاج Root (قادم في نسخة لاحقة)
